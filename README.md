# cookie-store-analysis
Power BI project analyzing cookie store product performance

Cookie Store Product Performance Analysis

 Problem Statement

Analyze the performance of products, categories, and flavors to identify top-selling items, most profitable products, and weak performers.

Key Questions 

What are the top-selling products?

Which products generate the most profit?

Which flavors are underperforming?

Which product categories generate the highest revenue?

Data & Tools 

Dataset: Fact_Sales, Dim_Product, Dim_Customer, Dim_Store, Dim_Date

Tools: Power BI Desktop

Key Insights 

Milk Chocolate Cookie and Mini Chocolate Chip Cookie had the highest quantity sold.

Triple Chocolate Cookie and Lotus Biscoff Cookie generated the highest profit.

Products in the Premium category generated the highest revenue, while products in the Healthy category had the lowest revenue.

Hazelnut and Cinnamon Oat flavors had the lowest quantity sold.

 Visuals Dashboard 

Page includes Bar Charts and Column Charts showing product sales, profit, category performance, and flavor performance, along with interactive slicers for Store and Date.
